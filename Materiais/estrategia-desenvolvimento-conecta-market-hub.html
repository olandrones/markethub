O Desafio

Por que este projeto existe

Três problemas raiz sustentam a ineficiência no sourcing internacional China → Brasil.

🔍

Cadeia Opaca

No Alibaba, a maioria dos anunciantes são trading companies — não fabricantes reais. Identificar fabricantes exige conhecimento especializado e buscas manuais via LLM.

70-80% são intermediários
📝

Processo Artesanal

O sourcing é operado manualmente via Excel, WhatsApp e e-mail. Nenhuma etapa — da pesquisa ao pagamento — está digitalizada de forma integrada.

1-2 semanas por produto
📈

Sem Viabilidade Integrada

Não existe um fluxo automatizado que conecte preço FOB, custos logísticos, impostos e margem final em um único estudo de viabilidade.

0% de integração

Regras de Negócio nos Fluxos

9 regras codificadas que governam os gateways e decisões do processo TO-BE.

RN1

Aprovação Automática

Range 25-35%: auto-aprovação. Fora: aprovação manual dos admins.

RN2

Pagamento via Despachante

Wise só inicia após despachante aprovar invoice na Logix.

RN3

Parcelas 30/70

30% na aprovação, 70% na prontidão. Splits configuráveis.

RN4

Chat IA Mediado

Cliente e CN nunca conversam diretamente. IA media tudo.

RN5

Fotos do Cliente

Fotos primárias são do cliente. Alibaba é complementar.

RN6

IA Fora do Cadastro

IA NÃO é usada no cadastro de catálogo. Apenas a partir do NCM.

RN7

Preço BR Obrigatório

Preço de compra no Brasil é pré-requisito para o pipeline.

RN8

NCM Human-in-the-Loop

IA sugere top 3-5 NCMs. Operador confirma. Penalidade: 1% VA.

RN9

Restrições Obrigatórias

Anuências + anti-dumping + alíquotas por NCM antes do sourcing.

Pipeline Completo

10 etapas do sourcing digitalizado

Do catálogo do cliente ao rastreamento da entrega — incluindo classificação NCM e análise de restrições antes da pesquisa de preços.

01

Catálogo automático

Cliente importa produtos via VTEX API, CSV ou cadastro manual. Enriquece com preço de compra no Brasil (obrigatório), fotos e características. IA não é usada nesta etapa (RN6).

02

Classificação NCM ia + human-in-the-loop

IA pré-classifica cada produto na tabela NCM (8 dígitos) via LLM econômico em batch. Se confiança < 90%, refinamento visual multimodal com foto. Operador confirma NCM entre top 3-5 sugestões. Classificação obrigatória — determina alíquotas e restrições (RN8).

03

Análise de Restrições automático

Com NCM confirmado, sistema verifica: anuências (Classif API, requer certificado digital), anti-dumping (CSV MDIC, 49 medidas contra China), alíquotas (IBPT API interim / Classif oficial). Alertas visuais: 🟢 Livre / 🟡 Requer licença / 🔴 Bloqueado (RN9).

04

Pesquisa Preços China ia + scraping

APIs de scraping buscam preços FOB no Alibaba. Tradução semântica PT→EN via LLM. Plataforma calcula diferença percentual entre preço brasileiro e FOB China.

05

Seleção decisão humana

Cliente visualiza comparativo de preços com alertas de restrições e seleciona quais produtos deseja importar. Decisão 100% humana — a plataforma informa, o cliente decide.

06

Sourcing ia core

IA busca fabricantes chineses reais usando specs do produto. Sistema gera RFQ bilíngue (EN + 中文) e envia ao Parceiro CN. Comunicação mediada — cliente e fornecedor nunca conversam diretamente (RN4).

07

Cotação semi-auto

Parceiro CN (Market Union) negocia presencialmente com fábricas e devolve preços FOB reais negociados via Portal do Parceiro.

08

Viabilidade (DDP) automático

Logix calcula custo total nacionalizado (DDP). Conecta Market Hub define margens. Aprovação automática se valor CN está entre 25%-35% do valor BR; fora do range, admins aprovam ou reprovam manualmente (RN1).

09

Aprovação automático

Cliente recebe proposta consolidada apenas com produtos aprovados, com estudo de viabilidade completo — preço FOB, custos, impostos, margem e preço final.

10

Pagamento + Rastreamento semi-auto

Invoice via empresa US. Pagamento só inicia após despachante aprovar invoice na Logix (RN2). 30% na aprovação, 70% na prontidão de envio (RN3, configurável). Via Wise API. Rastreamento integrado via Logix em tempo real.

Estratégia de Releases

Cronograma de 4 meses

Automação progressiva — a plataforma é operacional desde o R1, com o operador fazendo bridge manual enquanto os módulos são desenvolvidos.

Mês 1-2 • 8-10 semanas

R1 — Foundation

Base operacional da plataforma. Todos os portais funcionais com bridge manual do operador para etapas ainda não automatizadas.

✓ Catálogo com import VTEX/CSV/Manual
✓ Portal do cliente e dashboard operador
✓ Autenticação multi-tenant (Supabase RLS)
✓ E-mail automático de RFQ para Parceiro CN
✓ DDP manual/semi-auto (câmbio BCB)
✓ Classificação NCM manual (consulta Siscomex)
✓ Consulta manual restrições (Portal Único + MDIC)
✓ Dashboards básicos e tracking consumo
Entrega

Plataforma operacional — catálogo, seleção, RFQ por e-mail e viabilidade com input manual.

Mês 3 • 6-8 semanas

R2 — AI Engine + Portal CN

Motor de inteligência artificial ativado e parceiro chinês digitalizado. Classificação NCM, restrições, comparativo de preços e busca de fabricantes automatizados.

✓ Classificação NCM assistida por IA (pipeline 4 etapas)
✓ Análise automática restrições/anti-dumping
✓ Scraping Alibaba + comparativo BR vs CN
✓ Busca de fabricantes via LLM
✓ Portal do Parceiro CN (substitui e-mail)
✓ Chat IA mediado Cliente ↔ CN
Pré-req: Certificado digital e-CNPJ A1 (~R$ 200/ano) Pré-req: Registro IBPT com CNPJ Pré-req: Job sync CSV anti-dumping MDIC
Entrega

Inteligência artificial ativa — pesquisa de preços, NCM, restrições, fabricantes e comunicação mediada.

Mês 3-4 • 6-8 semanas

R3 — Financial + Logix

Pipeline financeiro automatizado. DDP manual substituído pela API Logix e pagamentos processados via Wise.

✓ Integração Logix API (DDP automático)
✓ Integração Wise (pagamentos 30/70)
✓ Invoices automáticas via empresa US
✓ Aprovação com range configurável (25-35%)
Pré-req: Reunião técnica com Logix Pré-req: Consulta legal (entidade US, scraping, NDA) Pré-req: Onboarding Wise Business API
Entrega

Pipeline financeiro completo — DDP automático, pagamentos internacionais e invoices integrados.

Mês 4 • 4-6 semanas

R4 — Scale

Plataforma completa e pronta para escalar. Tracking em tempo real, dashboards executivos, modo demo e recompra.

✓ Tracking real-time via Logix
✓ Dashboards completos e KPIs
✓ Alertas baseados em regras
✓ Modo demo para vendas
✓ Recompra facilitada
✓ Relatórios executivos
Entrega

Plataforma completa e escalável — pronta para 100+ clientes simultâneos.

O que acontece quando o caminho feliz não funciona? 7 cenários a definir no PRD.

01

a definir Cancelamento pelo cliente após pagamento de 30%

Quem assume o prejuízo? Busca fabricante alternativo automaticamente?

03

a definir Timeout de cotação do Parceiro CN

Prazo máximo para devolver cotação? Escalação automática? Outro parceiro acionado?

07

a definir API externa indisponível

Graceful degradation para scraping, LLM, Logix, Wise? Fila de retry? Notificação ao admin?

Fundação Técnica

Stack tecnológica

Tecnologias modernas, escaláveis e de mercado — escolhidas para performance, produtividade e manutenibilidade.

Frontend

Next.js + Tailwind

4 portais (cliente, operador, parceiro CN, admin) com componentes compartilhados e SSR.

Backend

Node.js + Fastify

API de alta performance com TypeScript end-to-end e gateway unificado.

Database

PostgreSQL (Supabase)

Multi-tenancy com Row Level Security. Realtime, auth e storage integrados.

Inteligência Artificial

Gateway Multi-Provider

KIMI, DeepSeek, Claude, GPT — roteamento por custo/qualidade com fallback automático.

Pagamentos

Wise API

Pagamentos cross-border USD/CNY/BRL com splits configuráveis (30/70).

Logística

Logix API (TOTVS)

DDP automático, rastreamento em tempo real e integração com despachante.

Queue System

BullMQ + Redis

Jobs assíncronos para scraping, chamadas LLM e integrações com retry e backoff.

Hosting

Vercel + Supabase

Deploy serverless com CDN global. Edge Functions para operações de baixa latência.

Integrações

Sistemas externos conectados à plataforma, classificados por criticidade.

VTEX/Bemol — Catálogo Alibaba — Scraping APIs KIMI API — Busca Fabricantes Logix/TOTVS — DDP + Tracking Wise — Pagamentos Internacionais Classif API — NCM + Restrições (certificado digital) MDIC — Anti-dumping (CSV, não API) IBPT — Alíquotas (interim, aproximado) BCB — Câmbio USD/BRL LLM Multi-Provider — Gateway IA
Nota sobre fontes de alíquotas

A IBPT API fornece alíquotas aproximadas (metodologia própria do instituto) — é fonte interim até o certificado digital estar disponível. A Classif API (Portal Único Siscomex) é a fonte oficial da Receita Federal, mas requer certificado digital ICP-Brasil (~R$ 200/ano). Diferenças de 2-3% possíveis entre IBPT e dados oficiais.
